

# Concierge One — Bulgaria Landing Page (Single Page)

## Overview
Build a single, self-contained luxury landing page showcasing Bulgaria as a travel destination. The page faithfully recreates the original design with enhanced interactions, all on one scrollable page — ready to be exported and embedded into a WordPress site.

## Design System
- **Colors**: Dark green (#022C22) background, gold (#D4AF37) accents, cream (#FDFBF7) for light sections
- **Fonts**: Playfair Display for headings, Plus Jakarta Sans for body text
- **Style**: Glassmorphism overlays, hover zoom on images, smooth scroll between sections

## Sections (Top to Bottom)

### 1. Sticky Navigation Bar
- Logo: "CONCIERGE**ONE**" in gold accent
- Nav links: Destinations, Villas, Experiences (smooth-scroll anchors)
- "Book VIP" gold CTA button
- Mobile hamburger menu with slide-in drawer

### 2. Hero Section
- Full-screen background image with dark gradient overlay
- Floating animated badge: "Direct Flights from DXB/AUH"
- Large headline: "The Hidden Gem of Europe"
- Two CTA buttons: "Explore Packages" (scrolls to packages section) and "Plan My Trip" (WhatsApp link)
- Bottom stats bar with glassmorphism: 5h Flight · 24°C Summer · Halal Food · Schengen Visa

### 3. Destinations — "Select Your Vibe"
- Horizontally swipeable on mobile, 4-column grid on desktop
- Four cards: Sofia, Plovdiv, Bansko, Katarino (with "TOP PICK" badge)
- Each card has background image, hover zoom, title, short description, and external link

### 4. VAIA Heights Villas
- Split layout: text left, image right (stacked on mobile)
- Feature icons grid: Private, Thermal Pools, Mountain Views, Private Chef
- "Download Brochure" CTA linking to external PDF
- Guest testimonial glassmorphism card overlaying the image

### 5. Photo Gallery
- Grid of villa and destination photos (6-8 images)
- Clicking any image opens a full-screen lightbox modal
- Lightbox has prev/next navigation arrows and close button
- Swipe support on mobile within the lightbox

### 6. Curated Adventures — Experiences
- Bento-grid layout with mixed card sizes
- Four experiences: Forest Safari (large), Hunting, Horse Riding, Mineral Healing (wide)
- Hover effects with image zoom and gradient overlays

### 7. Packages Overview
- 3 package cards side by side (stacked on mobile): e.g., "Sofia & Culture," "Mountain Retreat," "Wellness Escape"
- Each card shows: destination image, trip name, duration, key highlights, and a WhatsApp "Book Now" CTA

### 8. Final CTA / Contact
- "Ready for the Green Escape?" heading
- Large WhatsApp CTA button
- Direct phone number displayed

### 9. Floating WhatsApp Button
- Fixed bottom-right corner on all scroll positions
- Green WhatsApp icon with expand-on-hover "Chat Now" label

## Interactions & Enhancements
- **Photo Lightbox**: Reusable modal component with keyboard navigation (arrow keys, Escape to close)
- **Smooth Scroll**: All nav links and CTAs scroll smoothly to their target sections
- **Hover Effects**: Image scale on cards, button glow and scale, gold underline animations
- **Glassmorphism**: Frosted glass on stats bar, testimonial card, and navigation
- **Fully Responsive**: Mobile-first with swipeable carousels, stacked grids, and touch-friendly lightbox
